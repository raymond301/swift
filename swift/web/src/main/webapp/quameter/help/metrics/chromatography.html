The Metrics that Describe the Chromatography:
  • Peak Width of a chromatographic peak is the peak's full width at half maximum.
  • Peak Width Variability is a metric that describes the variation of peak widths throughout the Retention Spread .
  • Peptides per Minute is the rate that peptides are detected.
